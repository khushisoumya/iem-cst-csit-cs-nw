# IEM CST CSIT CS CS(NW) Departmental Website

A modern, responsive website for the Department of Cyber Security, Computer Science Technology, Computer Science Information Technology, Computer Science (Network), Institute of Engineering & Management (IEM), Kolkata.

## Features

- Responsive Design
- Modern UI with Tailwind CSS
- Department Information
- Faculty Section
- Research Section
- Placements
- Admissions
- Contact Page
- Optimized Assets
- Custom Favicon

## Tech Stack

- React
- Vite
- Tailwind CSS
- React Router
- Lucide React

## Installation

```bash
git clone <repository-url>
cd <project-name>
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Preview

```bash
npm run preview
```
